Isabel's Spa - Web Application
Overview

Isabel's Spa is a full-stack interactive web application built to establish a dedicated online presence for a local spa business (isabelspa.org). The platform transitions the business from manual scheduling to an automated, API-driven booking system, significantly improving client web accessibility and reducing administrative overhead for the business owner.
Key Features

    Automated Booking System: Integrates third-party scheduling APIs to streamline appointment creation, reducing booking steps by 50%.

    Administrative Optimization: Replaces manual tracking with an automated pipeline, saving the business owner an estimated 4+ hours of administrative work per week.

    Responsive Design: Translates specific client requirements into a functional, user-friendly interface accessible across multiple devices.

Tech Stack

    Frontend: HTML, CSS, JavaScript

    Backend: Node.js

    Integrations: REST APIs for automated scheduling
